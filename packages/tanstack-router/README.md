# @sdui-kit/tanstack-router

TanStack Router route helpers for SDUI Kit.

## Install

```bash
npm install @sdui-kit/core @sdui-kit/react @sdui-kit/tanstack-router @tanstack/react-router
pnpm add @sdui-kit/core @sdui-kit/react @sdui-kit/tanstack-router @tanstack/react-router
yarn add @sdui-kit/core @sdui-kit/react @sdui-kit/tanstack-router @tanstack/react-router
bun add @sdui-kit/core @sdui-kit/react @sdui-kit/tanstack-router @tanstack/react-router
```

Use the matching renderer package for your UI framework. The command above shows the React path.

Includes helpers for creating SDUI route context and mapping SDUI route manifests to TanStack Router-like route options.

See the repository README and `docs/integrations/tanstack-router.md` for full usage examples.
