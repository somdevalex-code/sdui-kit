# @sdui-kit/next

Next App Router navigation adapter for SDUI Kit.

## Install

```bash
npm install @sdui-kit/core @sdui-kit/react @sdui-kit/next
pnpm add @sdui-kit/core @sdui-kit/react @sdui-kit/next
yarn add @sdui-kit/core @sdui-kit/react @sdui-kit/next
bun add @sdui-kit/core @sdui-kit/react @sdui-kit/next
```

This package expects Next to be installed by the host app.

Includes helpers for translating SDUI navigation actions to a Next-style router and for creating route context from pathname, params, and search params.

See the repository README and `docs/integrations/next.md` for full usage examples.
