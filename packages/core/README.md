# @sdui-kit/core

Framework-agnostic runtime for SDUI Kit.

## Install

```bash
npm install @sdui-kit/core
pnpm add @sdui-kit/core
yarn add @sdui-kit/core
bun add @sdui-kit/core
```

Includes protocol types, component registry helpers, expression evaluation, action execution, screen loading state, cache/data adapter contracts, and payload validation.
