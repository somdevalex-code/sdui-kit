# @sdui-kit/forms

Headless form runtime for SDUI Kit.

## Install

```bash
npm install @sdui-kit/core @sdui-kit/forms
pnpm add @sdui-kit/core @sdui-kit/forms
yarn add @sdui-kit/core @sdui-kit/forms
bun add @sdui-kit/core @sdui-kit/forms
```

Includes form state, validation, touched/error tracking, dynamic field state, and submit integration with SDUI actions.

See the repository README and `docs/guide/forms.md` for full usage examples.
