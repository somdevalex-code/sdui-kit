# @sdui-kit/vue

Vue adapter for SDUI Kit.

This package builds on [`@sdui-kit/core`](https://www.npmjs.com/package/@sdui-kit/core).

## Install

**npm**

```bash
npm install @sdui-kit/core @sdui-kit/vue
```

**pnpm**

```bash
pnpm add @sdui-kit/core @sdui-kit/vue
```

**yarn**

```bash
yarn add @sdui-kit/core @sdui-kit/vue
```

**bun**

```bash
bun add @sdui-kit/core @sdui-kit/vue
```

This package expects Vue 3 to be installed by the host app.

Includes Vue provider, renderer, registry helpers, action composables, and screen rendering helpers for app-owned Vue components.

See the repository README and `docs/integrations/vue.md` for full usage examples.
